CSS tableのヘッダを固定してスクロール

width, heightを指定する必要がある。

table {width:400px;}
thead {display:table;background-color:#fff;}
th, td {width:200px;}
tbody{overflow-y:scroll;display:block;height:200px;}

        <tr><th>head1</th><th>head2</th></tr>
    </thead>
