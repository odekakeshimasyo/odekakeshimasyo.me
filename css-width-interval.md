⧸⧸CSS 等間隔に横並び
===

tableタグ、tdタグの要素にする

``` css
.interval {
	display:table;
	table-layout: fix;
	width:1000px;
}
.interval li {
	display:table-cell;
	border: 1px solid red;
}
```

``` html
<ul class="interval">
<li>111</li>
<li>222</li>
<li>333</li>
</ul>
```
